<p><ul><li>Binocular Stereo Vision</li><li>3D Structure Light</li><li>TOF</li></ul></p>
<p><p>Binocular stereo vision uses two cameras to capture an object from different perspectives, creating a 3D image based on the disparity between the two images. 3D structure light systems project patterns onto an object and measure the distortion to create a 3D image. Time-of-flight (TOF) systems emit light pulses and measure the time it takes for the light to bounce back, calculating the distance to objects in the scene. Each type offers unique advantages for various applications in industries such as automotive, healthcare, and robotics.</p></p>
